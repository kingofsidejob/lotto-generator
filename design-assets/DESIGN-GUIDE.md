## 📐 디자인 도구 추천

### OG 이미지 제작
1. **Figma** (무료) - https://figma.com
2. **Canva** (무료/유료) - https://canva.com
   - 템플릿: "Facebook Post" (1200×630) 선택
3. **Adobe Photoshop** (유료)

### Favicon 제작
1. **Figma** → Export as PNG/ICO
2. **RealFaviconGenerator** - https://realfavicongenerator.net/
   - 하나의 큰 이미지(512×512)를 업로드하면 모든 사이즈 자동 생성
3. **Favicon.io** - https://favicon.io/
   - 텍스트, 이모지, 이미지에서 favicon 생성

## 💡 디자인 팁

### OG 이미지
1. **텍스트 가독성 최우선**: 모바일에서도 읽기 쉬워야 함
2. **과한 디테일 지양**: 썸네일 크기에서도 명확해야 함
3. **브랜드 일관성 유지**: 모든 OG 이미지가 같은 스타일
4. **여백 충분히**: 텍스트 주변에 최소 40px 여백

### Favicon
1. **극도의 단순화**: 16×16에서도 인식 가능해야 함
2. **대비 강조**: 배경색과 아이콘 색의 명확한 대비
3. **테스트 필수**: 실제 브라우저 탭에서 확인
4. **다크 모드 고려**: 필요시 별도 다크 모드 버전 제작
